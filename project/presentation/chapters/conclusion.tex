\section{Conclusions \& Future Work}
\begin{frame}{Conclusions \& Future Work}
    \begin{itemize}
        \item \textbf{Cloud-Native transformation:} evolved a local prototype into a resilient, automated platform on Kubernetes.
        \item \textbf{Full GitOps automation:} a single \texttt{git push} flows to the cluster via Jenkins and ArgoCD -- no manual scripts.
        \item \textbf{Resilience by design:} Claim Check, Retry, Circuit Breaker, Idempotency and Stateless patterns make the system fault-tolerant.
        \item \textbf{Observability \& SLAs:} Prometheus/Grafana, autoscaling, and SLA tiers derived from real load tests.
        \item \textcolor{HUAblue}{\textbf{Future work:}} centralized identity with Keycloak (SSO), Ingress with TLS, and a custom domain for production-grade access.
    \end{itemize}
\end{frame}
